메뉴: 아래와 같이 router-link 로 링크를 할 수 있으며, 또는 a href=... 로 링크를 할 수 있다.
<router-link to=...> 는 Vue.js 가 A 태그로 변환한다. 즉, DOM 에는 router-link 가 아닌 A 태그로 변환되어져 있다.
아래 처럼 A 태그의 href 에 직접 링크를 주어서 SEO 로 동작하게 할 수 있다. route-link 는 DOM 에서만 A 태그로 변환될 뿐, HTML 에서는 그대로 router-link 로 표기되어 검색 로봇이 인식을 하지 못한다.
Go to Foo Go to Bar Go to user Go to user (2)
router-view 는 URL 에 매치되는 컴포넌트가 보인다. 메뉴는 일반 HTML 로 디자인하고, router 이동에 따라 해당 컴포넌트를 router-view 에 보여주면 된다. 이 때, 첫 페이지 또는 게시글 페이지에서 검색엔진에 색인시키고자 하는 내용을 div.main 안에 넣어주면 된다. 검색 로봇은 Vue.js 코드를 인식하지 못하는데, 웹 브라우저가 Vue.js 를 로드 할 때, created() 에서 div#seo 을 없애주면 된다.
